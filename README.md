# HairFlow 💈

Sistema de reserva de citas online para peluquerías, compuesto por tres aplicaciones:

## 📱 [HairFlow Mobile](https://github.com/emmanuelsf2/HairFlowMobile)
App móvil desarrollada en Kotlin para que los clientes reserven citas desde el móvil.

## 🖥️ [HairFlow Desktop](https://github.com/emmanuelsf2/HairFlowDesktop)
Aplicación de escritorio desarrollada con WPF para la gestión del salón.

## 🌐 [HairFlow Web](https://github.com/emmanuelsf2/HairFlowWeb)
Versión web para reservar citas desde el navegador.

## Tecnologías
- Kotlin / Jetpack Compose (móvil)
- C# / WPF (escritorio)
